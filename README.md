# HydrogenOrg Wastewater Regeneration Simulator v0.1

Conceptual Streamlit research prototype for wastewater treatment, water recovery, energy integration and resource recovery.

## Concept

Wastewater Input -> Treatment Process -> Water Recovery -> Energy Integration -> Hydrogen and Nutrient Recovery -> Environmental Impact

## Included layers

- Layer 1: Wastewater input characterization
- Layer 2: Treatment process simulation
- Layer 3: Water recovery balance
- Layer 4: Energy integration and conceptual hydrogen recovery
- Layer 5: Sludge and environmental impact indicators

## Run locally

```cmd
pip install -r requirements.txt
streamlit run app.py
```
